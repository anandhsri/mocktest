## Important: Git User vs GitHub Username

⚠️ **Note:** The git `user.name` is different from your **GitHub username**.

- **Git user.name**: Used for commit author (currently: "Anand Srinivasan")
- **GitHub username**: Your login username on GitHub.com (e.g., "anandsrinivasan" or similar)

## How to Find Your GitHub Username

### Method 1: Check GitHub Profile
1. Log into: **https://github.com**
2. Click your profile picture (top-right)
3. Your username is displayed in the dropdown

### Method 2: Check Settings
Visit: **https://github.com/settings/profile**
- Your username is shown at the top

### Method 3: Check URL
When you're logged into GitHub, your profile URL is:
```
https://github.com/YOUR_USERNAME
```

## Your Repository URL Format

Once you create the repository, your URL will be:

```
https://github.com/YOUR_GITHUB_USERNAME/neet-mock-test
```

**Example:**
- If your GitHub username is `anandsrinivasan`, your URL would be:
  ```
  https://github.com/anandsrinivasan/neet-mock-test
  ```

## Quick Setup Commands

Replace `YOUR_GITHUB_USERNAME` with your actual GitHub username:

```bash
cd /Users/anandsrinivasan/neet-mock-test

# Add remote (replace YOUR_GITHUB_USERNAME)
git remote add origin https://github.com/YOUR_GITHUB_USERNAME/neet-mock-test.git

# Push to GitHub
git push -u origin main
```
